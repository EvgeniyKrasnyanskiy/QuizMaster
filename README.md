# QuizMaster 🎓

Профессиональная мобильная платформа для проведения тестов и викторин. Приложение позволяет учителям создавать тесты и публиковать их в облако, а ученикам — проходить их и отправлять отчеты.

## 🚀 Основные возможности
- **Режим ученика**: Подписка на учителей, прохождение тестов, автоматическая генерация отчетов.
- **Режим учителя**: Встроенный редактор тестов, управление конфигурацией приложения, публикация в GitHub.
- **Безопасность**: Шифрование контента тестов (AES) и защита от повторного прохождения (кулдаун).
- **Облако**: Синхронизация через GitHub-репозитории (Master Data).

## 🛠 Технологический стек
- **Framework**: Expo (React Native)
- **Architecture**: Local Prebuild (Android Gradle)
- **Storage**: AsyncStorage + FileSystem
- **Cloud**: GitHub API / Raw Content

## 📦 Сборка приложения (Local Android)

Проект переведен на локальную сборку. Для создания APK выполните:

1. **Синхронизация версии**:
   ```powershell
   npm version patch  # Это обновит версию и запустит скрипт синхронизации
   ```
2. **Нативная подготовка**:
   ```powershell
   npx expo prebuild --platform android
   ```
3. **Компиляция**:
   ```powershell
   cd android
   ./gradlew assembleRelease
   ```
*APK будет доступен в: `android/app/build/outputs/apk/release/app-release.apk`*

## 📁 Структура данных
Приложение использует два типа репозиториев:
1. **QuizMaster**: Основной код приложения.
2. **quiz-master-data**: Репозиторий с данными (тесты, реестр, глобальный конфиг).

## 📝 Разработка
- **Константы**: `src/constants/appConfig.js`
- **Утилиты**: `src/utils.js`
- **Версионирование**: Управляется через `package.json` -> `scripts/sync-version.js`.
