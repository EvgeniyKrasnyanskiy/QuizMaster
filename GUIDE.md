# Руководство QuizMaster (Справка)

Это справочный файл, описывающий логику работы приложения, структуру данных и механизмы взаимодействия с облаком.

## 1. Структура приложения
Приложение разделено на два основных режима:
- **Режим ученика**: Прохождение тестов, получение результатов, подписка на учителей.
- **Режим учителя**: Создание и редактирование тестов, публикация в облако (GitHub), управление конфигурацией.

## 2. Работа с файлами и шифрование
Все тесты хранятся в формате `.dat`.
- **Шифрование**: Содержимое файла шифруется с помощью алгоритма AES (или его аналога в `src/utils.js`) с использованием ключа приложения (`APP_SALT`).
- **Безопасность**: Без ключа приложения прочитать содержимое теста невозможно. Это защищает вопросы от подглядывания ответов.

## 3. Облако и синхронизация (GitHub)
Приложение использует GitHub как сервер хранения.
- **Реестр (`registry.json`)**: Файл со списком всех тестов, их ID, названиями и количеством вопросов.
- **Папка `tests/`**: Здесь лежат сами `.dat` файлы.
- **Подписки**: Ученик подписывается на `Username` и `Repo` учителя. Приложение скачивает реестр этого учителя и отображает его тесты в библиотеке ученика.

## 4. Механизм обновлений
- Приложение проверяет файл `package.json` в репозитории `quiz-app-data`.
- Если версия в `package.json` на GitHub отличается от версии в приложении, пользователю предлагается скачать `app-release.apk`.
- **Процесс обновления**: Собрать APK -> Залить на GitHub как `app-release.apk` -> Изменить версию в `package.json`.

## 5. Инструменты разработчика (Режим учителя)
- **Сброс блокировок**: Удаляет историю прохождений. Позволяет заново проходить тесты, у которых еще не вышел срок блокировки (кулдаун).
- **Сброс кулдауна синхронизации**: Заставляет приложение принудительно обновить список тестов из облака при следующем входе в библиотеку.
- **Синхронизация конфига**: Загружает глобальные настройки (название приложения, цвета, лимиты) из файла `quiz-config.json` на GitHub.

## 6. Локальные папки (на устройстве)
- `Quizzes/Student`: Тесты, которые ученик загрузил вручную.
- `Quizzes/Downloads`: Тесты, автоматически скачанные из облака учителей.
- `Quizzes/Teacher`: Тесты, созданные учителем в редакторе.
